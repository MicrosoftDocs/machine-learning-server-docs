 #`getRemoteWorkspace`: Copy the workspace if the remote R session.

 Applies to version 1.1.0 of package mrsdeploy.
 
 ##Description
 
Copy all objects from the remote R session and load them into the local R session.
 
 
 ##Usage

```   
  getRemoteWorkspace()
 
```
